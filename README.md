## Demo Shopping Site

Full Stack Shopping Application

- Prisma with PostgressQL DB
- Images are saved & fetched from CloudinaryAPI
- Next.JS
- Sass/Tailwind
- Designed in Figma
- Some Photos with Photoshop
- GSAP/Framermotion Animations
- Utilizing Page Transition API
